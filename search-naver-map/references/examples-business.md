# Examples: Local-Business Uses

The same capabilities can support public-information research for a business owner:

- check where a Place appears for a local keyword;
- inspect public category, address, hours, links, menus, photos, and review counts;
- compare public review themes or owner-reply coverage;
- inspect which competitors expose Naver Booking and which products/time slots appear;
- track changes using saved normalized outputs.

These are examples, not an owner-specific mode. The skill does not encode competitive strategy, decide what a business should do, or post changes/replies on the owner's behalf.
